Create a modern, premium, Apple-style e-commerce website for India that sells daily-use products, electronics, gadgets, and home appliances.

🔹 Design & UI:
- Clean, minimal, premium UI inspired by Apple’s website (apple.com)
- Smooth scrolling animations where each product section reveals itself as the user scrolls
- Use large product images, bold typography, and white/neutral background
- Add subtle transitions, parallax scrolling, and fade-in effects
- Fully responsive design (mobile + tablet + desktop)

🔹 Homepage:
- Hero section with featured product (full-screen banner)
- Multiple scrollable product sections (like Apple product showcase)
- Each section should include:
  - Product image (high quality)
  - Product name
  - Short tagline
  - Brief description that appears on scroll
  - “Buy Now” and “Learn More” buttons

🔹 Categories:
Include sections for:
- Daily Essentials (groceries, personal care)
- Electronics (mobiles, laptops)
- Gadgets (smartwatches, earbuds)
- Home Appliances (AC, fridge, washing machine)

🔹 Product Page:
- Individual product page with:
  - Product images (carousel/gallery)
  - Detailed description
  - Specifications
  - Price in INR (₹)
  - Ratings & reviews
  - Add to Cart button
  - Related products section

🔹 Cart & Checkout:
- Shopping cart page with product summary
- Quantity update & remove option
- Price breakdown (subtotal, GST, delivery fee)

🔹 Payment Page (India-focused):
- Payment options:
  - UPI (Google Pay, PhonePe, Paytm)
  - Debit/Credit Cards
  - Net Banking
  - Cash on Delivery (COD)
- Address form (Name, Phone, PIN code, State, City)
- Order summary before payment
- Confirmation page after successful order

🔹 Backend (if included):
- Use Node.js / Firebase / Supabase for backend
- Store products, users, and orders
- Authentication (login/signup with OTP/email)

🔹 Tech Stack:
- Frontend: React.js / Next.js
- Styling: Tailwind CSS
- Animations: Framer Motion or GSAP
- Payment Integration: Razorpay or Stripe India

🔹 Extra Features:
- Search bar with suggestions
- Filters (price, category, brand)
- Wishlist functionality
- Dark/light mode toggle
- Fast loading and SEO optimized

🔹 Branding:
- Create a modern brand name and logo
- Use Indian currency and localization
- Ensure fonts and spacing feel premium like Apple
